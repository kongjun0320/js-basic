# Debouncing & Throttling

**防抖和节流都是用来控制某个函数在一定时间内执行多少次的技巧。**

当我们给 DOM 绑定事件的时候，加了防抖和节流函数特别有用。为什么呢？因为我们在事件和函数之前加了一个控制层。记住，我们是无法控制 DOM 事件触发频率的。

## 防抖(Debounce)

防抖技术可以把多个顺序地调用合并成一次

假想一下，你在电梯中，门快要关了，突然有人要上来。电梯并没有改变楼层，而是再次打开电梯门。电梯延迟了改变楼层的功能，但是优化了资源。
